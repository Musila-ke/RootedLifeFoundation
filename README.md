# Rooted Life Foundation

Website for the Rooted Life Foundation — a nonprofit advancing health equity for sickle cell warriors through education, support, advocacy, and environmental action.

Built with Next.js (App Router), TypeScript, Tailwind CSS v4, Motion, and GSAP.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animation**: Motion (React), GSAP
- **Fonts**: Inter, Playfair Display (next/font)
- **Logo**: [path]/public/logo.png
